# DermaPlus Clinic

## Descripcion del Proyecto

**DermaPlus** es una solucion web moderna para una clinica estetica que ofrece tratamientos faciales, corporales y dermatologicos. El proyecto digitaliza la captacion de clientes a traves de:

- **Landing Page**: Diseno atractivo para captar leads con formulario de registro
- **Pagina de Contacto**: Informacion de la clinica y formulario de consultas
- **Dashboard Interno**: Panel con graficas y estadisticas para el personal

## Tecnologias Utilizadas

- Next.js 14.2
- Chakra UI 2.8
- TypeScript 5.4
- Framer Motion
- ECharts
- Lucide React

## Instalacion y Uso

### Prerrequisitos

- Node.js 18.x o superior
- npm, yarn o pnpm

### Instalacion

```bash
# Clonar el repositorio
git clone https://github.com/Jaug18/PruebaTecnicaTwilbox-DermaPlus.git

# Navegar al directorio
cd dermaplus-clinic

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

### Scripts Disponibles

```bash
npm run dev      # Servidor de desarrollo en http://localhost:3000
npm run build    # Build de produccion
npm run start    # Iniciar servidor de produccion
npm run lint     # Ejecutar linter
```
